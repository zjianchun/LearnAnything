📚 三大核心法则
法则名称公式记忆口诀
同底数幂相乘aᵐ·aⁿ = aᵐ⁺ⁿ底不变，指数加
同底数幂相除aᵐ÷aⁿ = aᵐ⁻ⁿ底不变，指数减

🔑 核心记忆：乘→加，除→减，幂的幂→乘
🔢 例题精讲

计算：(x²y³)⁴ ÷ (x³y²)²

第一步：展开括号内的幂

= x^(2×4) · y^(3×4) ÷ (x^(3×2) · y^(2×2)) = x⁸y¹² ÷ x⁶y⁴

第二步：同底数幂相除

= x^(8-6) · y^(12-4) = x²y⁸

知识模块二：整式乘法与乘法公式

▼
🧐 几何视角

用面积模型理解多项式乘法：边长为(a+b)的正方形面积 = a² + ab + ab + b² = a² + 2ab + b²。完全平方公式有了几何直觉！

🌟 两大乘法公式
📌 平方差公式：(a+b)(a-b) = a² - b²
📌 完全平方公式：(a±b)² = a² ± 2ab + b²
🔢 多项式乘多项式 FOIL法

(x + 2)(x + 3) = ?

F: x·x = x² O: x·3 = 3x I: 2·x = 2x L: 2·3 = 6 合并：x² + 5x + 6
🔢 公式应用示例

计算 103 × 97（用平方差公式）

= (100+3)(100-3) = 100² - 3² = 10000 - 9 = 9991

✅ 巧妙利用公式，避免繁琐竖式计算！

⚠️ 错误诊断：完全平方陷阱

错误：(x+3)² = x² + 9

正确：(x+3)² = x² + 6x + 9（少了中间项 2×x×3=6x）

记住：完全平方 = 首平方 + 2倍交叉积 + 末平方

知识模块三：因式分解

▼
🧐 与整数分解类比

整数因式分解：12 = 2×2×3。多项式也可以！x²-4 = (x+2)(x-2)。这就像把一个数"拆开"——因式分解是乘法展开的逆运算。

🛠️ 两大基本方法
方法一：提取公因式法

找到各项共同的因数，提取到括号外

例：6x²y - 9xy² + 3xy 公因式：3xy = 3xy(2x - 3y + 1)

步骤：①找公因式 ②提出 ③括号内=原式÷公因式

方法二：公式法（逆用乘法公式）
平方差：a² - b² = (a+b)(a-b) 完全平方：a² + 2ab + b² = (a+b)² a² - 2ab + b² = (a-b)²

例：分解 4x² - 9

= (2x)² - 3² = (2x+3)(2x-3)
🔄 交互练习：判断方法
Q：对 x² - 6x + 9 进行因式分解，应用哪个公式？
Q：因式分解 2a²b - 4ab² = ?
⚠️ 因式分解常见错误

错误：x² - 4 = (x-2)²

正确：x² - 4 = (x+2)(x-2)（这是平方差，不是完全平方！）

检验方法：分解后展开验证是否等于原式

💡 实战问题

一块正方形地砖边长为(x+2)cm，中间挖去一个边长为(x-2)cm的正方形空洞。求剩余部分的面积（先展开再化简，然后因式分解）

解题过程（点击查看）
大正方形面积：(x+2)² = x² + 4x + 4 小正方形面积：(x-2)² = x² - 4x + 4 剩余面积 = (x²+4x+4) - (x²-4x+4) = 8x 因式分解：8x = 8×x

结论：剩余面积 = 8x cm²（正比于边长x，很有规律！）
