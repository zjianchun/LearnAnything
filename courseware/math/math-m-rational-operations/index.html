📖 模块一：有理数的加法与减法
📖 AND（背景）

在小学，我们学的加减法都是正数之间的运算。现在有了负数，情况复杂了——方向不同的两个数相加，结果应该是多少？

⚡ BUT（冲突）

比如温度：早上-3°C，下午升高5°C，现在是几度？用加法：(-3) + 5 = ?  答案不是 -3+5=2（正确），也不是8（错误）。符号让计算变复杂了。

💡 THEREFORE（解决）

用"同号相加取同号、异号相加取大号符号、与零相加不变"三条规则，统一处理所有加法。

➕ 加法法则

  • 同号相加：取相同符号，绝对值相加
  • 异号相加：取绝对值大的符号，绝对值相减
  • 加零：任何数加0等于本身
  • 互为相反数：和为0

➖ 减法法则

  • 减去一个数等于加上它的相反数
  • a − b = a + (−b)
  • 这样减法变加法，用加法法则计算
  • 例：3 − (−5) = 3 + 5 = 8
📐 例题演示
计算 (-7) + 3
步骤1：异号相加，取绝对值大的(-7)的符号，即负号
步骤2：绝对值相减：|-7| − |3| = 7 − 3 = 4
结果：(-7) + 3 = -4
📐 例题演示（减法转加法）
计算 (-5) − (-8)
步骤1：减去(-8)等于加上+8，即 (-5) + 8
步骤2：异号相加，取|8|的符号（正），|8|−|5|=3
结果：(-5) − (-8) = (-5) + 8 = 3
⚠️ 反例镜——常见错误

错误：3 − (−5) = 3 − 5 = −2（忘记减负变加正）

正确：3 − (−5) = 3 + (+5) = 8
记忆口诀："减负变加正"，两个负号相遇变正号

📖 模块二：有理数的乘法与除法
💡 THEREFORE（解决）

乘除法的核心是符号法则：同号得正，异号得负。绝对值按普通乘除法计算。

运算符号规律举例
(+)×(+)正×正=正3×4=12
(+)×(−)正×负=负3×(−4)=−12
(−)×(+)负×正=负(−3)×4=−12
(−)×(−)负×负=正(−3)×(−4)=12

📌 多个因数相乘的符号判断

数一数负数的个数：

  • 负数个数为偶数 → 乘积为正
  • 负数个数为奇数 → 乘积为负
  • 有一个因数为0 → 乘积为0
🔬 放大镜——为什么"负负得正"？

用规律推导：(−1)×3 = −3，(−1)×2 = −2，(−1)×1 = −1，(−1)×0 = 0，依照规律向下：(−1)×(−1) = +1！数学要保持一致性，所以负负必须得正。

另一个理解方式：(-3)×(-4) 可理解为"欠债3元的相反是存款3元，4次这样的相反就是存款12元"→ +12

📌 除法法则

除法：同号得正，异号得负（与乘法相同）

  • a ÷ b = a × (1/b)（除以一个数等于乘以它的倒数）
  • （−12）÷（−4）= +(12÷4) = 3
  • （−12）÷ 4 = −(12÷4) = −3

📖 模块三：乘方与混合运算
⚡ BUT（冲突）

注意：(−3)² 和 −3² 是一样的吗？很多同学在这里出错！

📌 乘方

  • a^n = a × a × … × a（n个a相乘）
  • 负数的偶次幂为正：(−2)² = (−2)×(−2) = 4
  • 负数的奇次幂为负：(−2)³ = −8
⚠️ 反例镜——括号的关键

区分 (−3)² 和 −3²：

(−3)² = (−3)×(−3) = 9（括号里的-3作为底数）
−3² = −(3²) = −(3×3) = −9（只有3作为底数，负号在外面）
结论：括号很重要！

📌 混合运算顺序

  1. 先算乘方
  2. 再算乘除
  3. 最后算加减
  4. 有括号先算括号内
📐 例题：(-2)³ + 3×(-4) − (-1)
步骤1（乘方）：(-2)³ = -8
步骤2（乘法）：3×(-4) = -12
步骤3（加减）：-8 + (-12) − (-1) = -8 + (-12) + 1
= -8 - 12 + 1 = -19
