模块一：弧长公式

🔔 ABT引入

你要给圆形蛋糕的一块（扇形）描边——And切了60°的一块，弧边有多长？；But60°只是360°的1/6，所以弧长是整个圆周的1/6；Therefore弧长 = (圆心角/360°) × 2πr，这就是弧长公式的直觉！

📖 弧长公式推导

圆心角n°对应的弧 = 整圆周长 × (n/360)
弧长 L = (n/360) × 2πr = nπr/180
其中 n 为圆心角度数，r 为半径

弧度制版本：若圆心角为α弧度，则弧长 L = αr
圆心角: 120° 半径: 80

模块二：扇形面积

🔔 ABT引入

披萨店要计算不同份额的披萨面积——And切成6份的披萨，每份面积是总面积的1/6；But如果圆心角不均匀，每份面积就按角度比例分配；Therefore扇形面积 = (圆心角/360°) × 圆面积！

📖 扇形面积公式

扇形面积 S = (n/360) × πr²
用弧长表示：
S = (1/2) × L × r（L为弧长）
验证：(1/2)×L×r = (1/2)×(nπr/180)×r = nπr²/360 ✓
圆心角: 90°

🔬 深度理解：公式间的联系（五镜法：联系镜）

三个公式的关系链：
圆心角n° → 弧长L = nπr/180
圆心角n° → 扇形面积S = nπr²/360
弧长L → 扇形面积S = Lr/2

记忆口诀："弧长半径积一半，就是扇形的面积"

🎯 综合任务：环形喷水池设计

某公园要设计一个扇形喷水池，圆心角为150°，外弧半径R=10米，内弧半径r=6米（环形扇形）。

🪜 脚手架步骤
  1. 计算外弧弧长（R=10，n=150°）
  2. 计算内弧弧长（r=6，n=150°）
  3. 计算整个环形扇形的面积（大扇形-小扇形）
  4. 计算铺设边缘围栏的总长度（外弧+内弧+两段直边）
解：
① 外弧 = (150/360)×2π×10 = 25π/3 ≈ 26.2m
② 内弧 = (150/360)×2π×6 = 5π ≈ 15.7m
③ 大扇形面积=(150/360)×π×100=125π/3；小扇形=(150/360)×π×36=15π
   环形扇形面积=125π/3-15π=125π/3-45π/3=80π/3≈83.8m²
④ 两段直边各=(R-r)=4m；围栏=25π/3+5π+4+4=25π/3+5π+8≈49.9m
