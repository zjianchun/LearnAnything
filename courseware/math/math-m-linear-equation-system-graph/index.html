📖 模块一：一次函数回顾
🎯 ABT引入：
And：你已经学过正比例函数 y=kx。
But：现实中很多问题的起点不是0——手机套餐有月租费，出租车有起步价……
Therefore：我们需要 y = kx + b，加上一个常数b，这就是更一般的一次函数！
y = kx + b（k≠0）
k：斜率（决定倾斜程度）　b：截距（图象与y轴的交点纵坐标）

k>0：从左下到右上

y随x增大而增大

k<0：从左上到右下

y随x增大而减小

🏋️ 即时练习：y = -2x + 5 的图象与 y 轴的交点是？

✖️ 模块二：图象法解方程组
🔑 核心思想：二元一次方程组的每个方程，都可以看作一次函数。
方程组的解 = 两条直线的交点坐标！
方程组 { y = x + 1
       { y = -x + 3

解 = 两直线交点 = (1, 2)

📊 三种情况对比（对比法）

✅ 唯一解

两直线相交
k₁ ≠ k₂

❌ 无解

两直线平行
k₁=k₂, b₁≠b₂

♾️ 无数解

两直线重合
k₁=k₂, b₁=b₂

🏋️ 即时练习：方程组 {y=2x+1, y=2x-3} 的解的情况是？

🔑 模块三：找交点坐标
🛠️ 方法：联立两函数方程，令 y₁=y₂，解 x，再求 y

例题：求 y=x+1 与 y=-x+3 的交点

步骤1：令两式相等

x + 1 = -x + 3

步骤2：解方程

2x = 2，x = 1

步骤3：代入求y

y = 1 + 1 = 2

结论：交点为 (1, 2)

🏋️ 即时练习：y = x + 2 与 y = -x + 4 的交点坐标是？

🎯 综合任务：联合定价问题

A商店：商品定价 y = 5x + 10（x为数量，y为总价元），B商店：y = 3x + 20。

🪜 解题脚手架

  1. 令两式相等：5x + 10 = 3x + 20，解出 x（即什么数量时两店价格相同）
  2. 代入求 y（相同时的价格）
  3. 当 x < 该值时，哪个商店更便宜？当 x > 该值时呢？
  4. （拓展）在图上标出交点，并分析两段的意义
