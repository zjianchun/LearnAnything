知识模块一：直接开平方法 & 因式分解法

▼
🧐 投篮问题

篮球从高度 h = -5t² + 10t + 2 处抛出（t秒后高度h米），什么时候落地（h=0）？需要解 -5t²+10t+2=0，二次方程！

🔑 方法一：直接开平方法

条件：能变成 (x+m)² = k 的形式

解方程 (x-3)² = 7

  • 两边开平方：x-3 = ±√7
  • x = 3 ± √7
  • 即 x₁ = 3+√7，x₂ = 3-√7
x = 3 ± √7 ≈ 3 ± 2.646
🔑 方法二：因式分解法

条件：左边能分解因式（右边=0）

解方程 x² + 5x + 6 = 0

  • 分解：(x+2)(x+3) = 0（找两数和为5，积为6：2和3）
  • 零积性质：x+2=0 或 x+3=0
  • 解得：x=-2 或 x=-3
x₁ = -2, x₂ = -3
⚡ 特殊情形：缺项方程
缺c项：x² - 5x = 0 → x(x-5)=0 → x=0或x=5 缺b项：2x² - 8 = 0 → x²=4 → x=±2

⚠️ 禁止两边都除以x！（x=0可能是根，会丢根！）

知识模块二：配方法

▼
🧐 为什么要配方？

有些方程不容易因式分解（比如 x²+2x-4=0），配方法把它"变形"成可以直接开平方的形式，是万能方法！

配方核心：x² + bx = (x + b/2)² - (b/2)²
🔧 配方法完整步骤

解方程 2x² + 4x - 6 = 0

  • 各项除以2（使x²系数为1）：x² + 2x - 3 = 0
  • 常数移到右边：x² + 2x = 3
  • 配方（加(2/2)²=1）：x² + 2x + 1 = 3 + 1
  • 整理为完全平方：(x+1)² = 4
  • 开平方：x+1 = ±2
  • x = 1 或 x = -3
x₁ = 1, x₂ = -3
⚠️ 配方常见错误

错误：配方时忘记在右边也加同样的数

x² + 2x = 3 → x² + 2x + 1 = 3（漏了右边+1）

正确：x² + 2x + 1 = 3 + 1（两边同时+1）

知识模块三：求根公式与判别式

▼
🧐 数学的终极武器

阿拉伯数学家花拉子米（9世纪）、印度数学家婆什伽罗（12世纪）逐步完善了这个公式。今天，它是解一元二次方程的万能钥匙！

ax² + bx + c = 0 (a≠0) 的根：

x = (-b ± √(b²-4ac)) / (2a)
🔍 判别式 Δ = b² - 4ac
Δ > 0
两个不相等实数根

x₁ ≠ x₂，均为实数

Δ = 0
两个相等实数根

x₁ = x₂ = -b/(2a)

Δ < 0
无实数根

在实数范围内无解

🔢 求根公式示例

解方程 2x² - 3x - 2 = 0

  • 识别系数：a=2, b=-3, c=-2
  • 计算判别式：Δ=(-3)²-4×2×(-2)=9+16=25>0，两实根
  • 代入公式：x=[3±√25]/(2×2)=[3±5]/4
  • x₁=(3+5)/4=2，x₂=(3-5)/4=-1/2
x₁ = 2, x₂ = -1/2

知识模块四：韦达定理（根与系数的关系）

▼
🧐 不需要求根的技巧

有时候我们不需要知道根的具体值，只需要知道两根之和与积。法国数学家韦达（16世纪）发现了这一规律！

若 x₁, x₂ 是 ax²+bx+c=0 的两根，则：
x₁ + x₂ = -b/a（两根之和）
x₁ · x₂ = c/a（两根之积）
🔢 韦达定理应用

已知 x²-5x+6=0 的两根 x₁, x₂，求 x₁²+x₂²

  • 由韦达定理：x₁+x₂=5，x₁·x₂=6
  • x₁²+x₂² = (x₁+x₂)² - 2x₁x₂
  • = 25 - 12 = 13
x₁² + x₂² = 13

🌟 拓展1：用二次方程解几何题

直角三角形斜边为10，两直角边之差为2，求两直角边的长。

设较短边为x，则：x² + (x+2)² = 100，解得 x=6，另一边=8（3:4:5的10倍）

🌟 拓展2：判别式的几何意义

y = ax² + bx + c 的图像是抛物线：

Δ > 0：与x轴有2个交点 | Δ = 0：有1个切点 | Δ < 0：不与x轴相交

🌟 拓展3：高阶挑战

解方程 (x²-x)² - (x²-x) - 6 = 0

令 t = x²-x，则 t²-t-6=0，(t-3)(t+2)=0，再解 x²-x-3=0 和 x²-x+2=0（后者Δ<0无解）
