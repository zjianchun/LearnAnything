📖 模块一：认识分式方程

And（问题引入）

🚗 小李开车从A到B，速度为 v km/h，距离60km，需要 60/v 小时。若速度增加20 km/h，时间减少1小时，建立方程：

60/(v+20) = 60/v - 1

注意分母中含有未知数 v！这就是分式方程。

But（与整式方程的区别）

类型特征例子
整式方程分母不含未知数2x + 3 = 7
分式方程分母含未知数1/x + 2 = 3/x

Therefore（核心挑战）

分式方程的分母不能为零！这是后面产生增根问题的根源。解题时必须检验解是否使原方程的分母为零。

⚙️ 模块二：解分式方程（去分母法）

And（核心思路）

分式方程难以直接求解，但如果两边同乘以最简公分母，就能把分式方程转化为整式方程来解！

Therefore（解题四步法）

例题：解 1/(x-2) + 1 = 3/(x-2)

找最简公分母（LCD）：分母为 (x-2)，LCD = x-2
两边乘以LCD，去分母：
(x-2) × [1/(x-2) + 1] = (x-2) × [3/(x-2)]
1 + (x-2) = 3
解整式方程：x - 1 = 3 → x = 4
验根（⚠️ 必须做！）：将 x=4 代入分母 (x-2)=2 ≠ 0
✓ 有效，x=4 是原方程的解
⚡ 解 2/x + 1 = 5/x，x = ？
⚠️ 模块三：增根问题（分式方程的特殊挑战）

And（什么是增根）

去分母后得到的整式方程的解，可能恰好使原分式方程的分母为零。这个解在整式方程中有意义，但在原分式方程中无意义，叫做增根，必须舍去！

But（增根示例）

例：解 (x+1)/(x-1) = 2/(x-1)

LCD = (x-1)，两边×(x-1)
x+1 = 2 → x = 1
验根：x=1 代入分母 (x-1) = 0！！分母为零！
✗ x=1 是增根，必须舍去！原方程无解。

Therefore（增根判断规则）

✅ 有效解

解 x = k 代入所有分母，分母均 ≠ 0

→ 保留该解

❌ 增根（舍去）

解 x = k 代入某个分母，使分母 = 0

→ 舍去该解，注明"原方程无解"

⚠️ 黄金法则：解分式方程必须验根！

不验根是解分式方程最严重的错误，会导致答案错误。每次都要检验解是否使分母为零。

🚀 综合任务：工程问题

任务：甲单独完成某项工程需要10天，乙单独完成需要15天。两人合作，完成后甲还需单独继续工作x天才能完成剩余部分，使整个工程恰好在8天内完成。求x。

🪜 解题脚手架

  1. 甲的效率：1/10（每天完成1/10）
  2. 乙的效率：1/15（每天完成1/15）
  3. 合作8天，甲前x天还有单独工作：设两人合作t天，甲再单独x天，t+x=8
  4. 工程总量=1：合作部分+甲单独部分=1
  5. 建立方程并求解（提示：设两人合作y天后，甲单独工作8-y天...）

【重建题目】甲独做10天，乙独做15天，合作几天完成？

✅ 简化版：甲乙合作几天完成整项工程？

设合作 x 天：x/10 + x/15 = 1

公分母30：3x + 2x = 30 → 5x = 30 → x = 6

合作6天完成。验根：x=6，分母10和15均≠0 ✓
