知识模块一：分式的概念与约分通分

▼
🧐 生活情境

用 x 升水平均分给 (x+3) 人，每人得 x/(x+3) 升。当 x=-3 时，分母为0，就没法分了！这就是为什么要限制分母不为0。

⚠️
分式有意义的条件：分母 ≠ 0
对分式 A/B，必须 B ≠ 0（A、B是整式，B含字母）
✂️ 分式的约分
约分原则：分子分母除以公因式（因式分解后消去公因式）
🔢 约分示例

约分 (x²-4)/(x²-x-2)

  • 分解分子：x²-4 = (x+2)(x-2)
  • 分解分母：x²-x-2 = (x-2)(x+1)
  • 消公因式 (x-2)：= (x+2)/(x+1)
原式 = (x+2)(x-2) / [(x-2)(x+1)] = (x+2)/(x+1) （x≠2, x≠-1）
🔢 通分
通分示例：将 1/(x+1) 和 2/(x-1) 通分
  • 找最简公分母（LCD）= (x+1)(x-1)
  • 第一个：1/(x+1) = (x-1)/[(x+1)(x-1)]
  • 第二个：2/(x-1) = 2(x+1)/[(x+1)(x-1)]
⚠️ 约分常见错误

错误：(x²+4)/(x+4) = x （直接消去4）

正确：分子 x²+4 和分母 x+4 无公因式，无法约分！

只有公因式（乘积关系）才能约分，加减关系的项不能消！

知识模块二：分式的四则运算

▼
🧐 类比分数运算

分式运算与分数完全类似：1/2 + 1/3 = 3/6 + 2/6 = 5/6。把数字换成代数式，道理不变！先通分，再运算。

运算规则示例
乘法分子×分子，分母×分母(a/b)×(c/d) = ac/bd
除法乘以倒数(a/b)÷(c/d) = (a/b)×(d/c) = ad/bc
同分母加减分母不变，分子加减a/c ± b/c = (a±b)/c
异分母加减先通分，再加减1/(x+1) + 1/(x-1) → 通分
🔢 加减运算示例

计算：1/(x+1) - 1/(x-1)

  • 公分母 = (x+1)(x-1) = x²-1
  • 通分：(x-1)/(x²-1) - (x+1)/(x²-1)
  • 分子相减：(x-1)-(x+1) = -2
  • 结果：-2/(x²-1)
= -2 / (x²-1) （x≠1, x≠-1）
🔢 乘除运算示例

计算：(x²-1)/(x+2) ÷ (x-1)/(x²-4)

  • 除法变乘法，乘以第二个分式的倒数
  • = (x²-1)/(x+2) × (x²-4)/(x-1)
  • 因式分解：x²-1=(x+1)(x-1)，x²-4=(x+2)(x-2)
  • 约分消公因式
= (x+1)(x-1)(x+2)(x-2) / [(x+2)(x-1)] = (x+1)(x-2)
⚠️ 分式加减错误

错误：1/x + 1/y = 2/(x+y)

正确：1/x + 1/y = y/(xy) + x/(xy) = (x+y)/(xy)

分母相加≠新分母！必须通分！

知识模块三：分式方程与增根

▼
🧐 为什么要验根？

解分式方程时，我们"去分母"——但如果某个解让分母=0，那它其实不合法！这样的"假解"叫增根，必须舍去。

解分式方程步骤：
①确定公分母 → ②两边乘公分母（去分母）→ ③解整式方程 → ④验根（代回检验分母是否为0）
🔢 完整示例：解方程

解：1/(x-1) + 2 = 3/(x-1)

  • 公分母 = (x-1)
  • 两边乘(x-1)：1 + 2(x-1) = 3
  • 展开：1 + 2x - 2 = 3 → 2x - 1 = 3 → x = 2
  • 验根：x=2 时，x-1=1≠0 ✅，分母不为0，x=2是合法解
∴ x = 2
⚠️ 含增根的示例

解：x/(x-2) = 2/(x-2) + 1

  • 两边乘(x-2)：x = 2 + (x-2) = x
  • 整理：0 = 0，说明原本是恒等式？
  • 实际上若直接解得 x=2，代入验证：x-2=0，分母为0 ❌
  • x=2 是增根，方程无解

拓展：进阶知识

🌟 拓展1：繁分式

分子或分母本身也是分式，叫繁分式。化简：(1 + 1/x) / (1 - 1/x)

= [(x+1)/x] / [(x-1)/x] = (x+1)/(x-1)

🌟 拓展2：分式方程的实际应用

追及问题：速度差、行程问题都涉及分式方程。

例：甲的速度是乙的1.5倍，乙走了x小时，甲走了相同路程需要 x/1.5 小时。

🌟 拓展3：与函数的联系

分式 y = k/x 是反比例函数（k≠0, x≠0）。理解了分式，就理解了反比例函数的定义域！
