📖 模块一：认识一元二次方程

And（从一次到二次）

你已经会解 2x + 3 = 7（一次方程，最高次数1）。现在面临一个新问题：

一块正方形土地面积为25m²，边长是多少？即：x² = 25

这里 x 的最高次数是 2！这就是一元二次方程。

But（为什么更难？）

一次方程只有一个解，而二次方程最多有两个解！x² = 25 → x = 5 或 x = -5。

Therefore（标准形式）

ax² + bx + c = 0   （a ≠ 0）

其中 a 是二次项系数，b 是一次项系数，c 是常数项。

例：将 2x² + 3x = 5 化为标准形式

→ 2x² + 3x - 5 = 0

a = 2，b = 3，c = -5

🔮 模块二：判别式 Δ = b² - 4ac

And（解的数量预判）

在用公式求解之前，我们可以先"预判"方程有几个实数解——这就是判别式 Δ（读作"Delta"）的作用。

Therefore（三种情况）

Δ = b² - 4ac
Δ > 0
两个不相等的实数根
x₁ ≠ x₂
Δ = 0
两个相等的实数根
x₁ = x₂（重根）
Δ < 0
没有实数根
（初中阶段无解）

🔭 示例计算

例：x² - 5x + 6 = 0（a=1, b=-5, c=6）

Δ = (-5)² - 4×1×6 = 25 - 24 = 1 > 0

→ 有两个不相等实数根 ✓

例：x² - 2x + 1 = 0（a=1, b=-2, c=1）

Δ = (-2)² - 4×1×1 = 4 - 4 = 0

→ 有两个相等实数根（重根）

🔢 模块三：求根公式法

And（解法的演变）

古人经过配方法推导，得到了一个通用公式，可以解任意一元二次方程！

Therefore（求根公式）

x = (-b ± √(b²-4ac)) / (2a)

其中 ± 表示"加或减"，所以有两个根 x₁ 和 x₂。

🔭 完整求解示例

解方程：2x² - 5x + 3 = 0

确定系数：a=2, b=-5, c=3
计算 Δ：Δ = (-5)² - 4×2×3 = 25-24 = 1 > 0，有两根
代入公式：x = (5 ± √1) / (2×2) = (5 ± 1) / 4
求两根：x₁ = (5+1)/4 = 6/4 = 3/2；x₂ = (5-1)/4 = 4/4 = 1
验算：代入原式：2×(3/2)²-5×(3/2)+3 = 9/2-15/2+3=0 ✓

🪞 错误诊断

❌ 常见错误：Δ = b² - 4ac，当 b 为负数时

b = -5：(-5)² = 25 ✓（注意加括号！）

错误做法：-5² = -25 ✗（符号算错）

✅ 永远对 b 加括号：(-b)² ！

🚀 综合任务：面积问题

任务：一个矩形花园，长比宽多3米，面积为40平方米。求花园的长和宽。

🪜 解题脚手架

  1. 设宽为 x 米，则长为 _____ 米
  2. 面积方程：x × _____ = 40
  3. 展开：x² + _____ = 40
  4. 化为标准形式：x² + _____ - 40 = 0
  5. 计算判别式 Δ = _____
  6. 用求根公式解出 x = _____（取正值）
  7. 长 = _____，宽 = _____
✅ 参考答案

设宽 x，长 x+3，方程：x(x+3)=40

x² + 3x - 40 = 0

Δ = 9 + 160 = 169，√169 = 13

x = (-3±13)/2 → x₁=5，x₂=-8（舍负）

宽=5米，长=8米。验算：5×8=40 ✓
