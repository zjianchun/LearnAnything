模块一：认识平面直角坐标系

🗺️ ABT引入：寻宝游戏的困境

And（背景）：小明在寻宝游戏中，地图说"从大树往东走5步，再往北走3步"。

But（冲突）：地图越来越复杂，"东南西北+步数"的描述让人晕头转向，还容易出错！

Therefore（转折）：数学家发明了平面直角坐标系——用有序数对 (x, y) 精确定位任意点！

📐 基本概念

在平面内建立两条互相垂直、有公共原点的数轴：

  • x 轴：水平数轴，向右为正方向 →
  • y 轴：竖直数轴，向上为正方向 ↑
  • 原点 O：两轴交点，坐标为 (0, 0)
  • 点 P 的坐标 (a, b)：a 是 x 坐标（横坐标），b 是 y 坐标（纵坐标）

⚠️ 常见错误诊断

❌混淆坐标顺序：写成 (y, x)
✅记忆口诀："先横(x)后纵(y)"——先写 x 坐标，再写 y 坐标
❌认为坐标轴上的点属于某个象限
✅坐标轴上的点不属于任何象限！只有"区域"内的点才属于象限

🔁 关于坐标轴的对称点规律

  • 关于 x 轴对称：(a, b) ↔ (a, −b)　 → y 坐标取反
  • 关于 y 轴对称：(a, b) ↔ (−a, b)　 → x 坐标取反
  • 关于 原点对称：(a, b) ↔ (−a, −b) → 两个坐标都取反

📏 到坐标轴的距离

  • 点 P(a, b) 到 x 轴的距离 = |b|
  • 点 P(a, b) 到 y 轴的距离 = |a|

拓展延伸

📜 笛卡尔的灵感

坐标系由法国数学家笛卡尔（Descartes）在17世纪发明，又称"笛卡尔坐标系"。传说他病床上观察天花板爬行的苍蝇，意识到用两个距离就能精确描述位置，从而奠定了解析几何的基础，将代数与几何完美统一！

🌐 三维坐标系

在 xOy 平面基础上再加一条 z 轴（垂直纸面），形成空间直角坐标系，用三个数 (x, y, z) 描述三维空间的位置。GPS 卫星定位、3D 建模、游戏引擎都依赖三维坐标！

💡 极坐标系

除了直角坐标，还有极坐标系：用（距离 r，角度 θ）描述点的位置，在物理、工程和导航领域广泛使用，是将来高中数学的重要内容。
