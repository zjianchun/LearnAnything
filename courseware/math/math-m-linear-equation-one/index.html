📖 模块一：认识一元一次方程

And（背景）

小明去超市买苹果，每千克5元，他买了若干千克后，共花了30元。这里有一个未知数——他买了多少千克？

But（问题）

我们不能直接看出答案，但我们可以用字母代替未知数，建立一个等量关系！

设买了 x 千克，则：

5x = 30

这就是一个一元一次方程！

Therefore（核心概念）

一元一次方程：只含有一个未知数，且未知数的最高次数为1的等式。

标准形式：ax + b = c（其中 a ≠ 0）

特征三要素：① 是等式 ② 只含一个未知数 ③ 未知数次数为1

🔭 深度理解：比较镜

表达式 是方程？ 是一元？ 是一次？ 结论
2x + 3 = 7 ✓ ✓ ✓ ✓ 是
x² = 4 ✓ ✓ ✗(2次) ✗ 否
x + y = 5 ✓ ✗(两个) ✓ ✗ 否

⚙️ 模块二：解方程的方法

And（天平模型）

想象一架平衡的天平。左边放了 2x 和 3，右边放了 9。天平平衡，说明两边相等。

But（变形规则）

要保持天平平衡，只要两边做相同的操作，等式就不变！

Therefore（等式性质）

性质1：加减
若 a = b，则 a ± c = b ± c
性质2：乘除
若 a = b，则 a × c = b × c（c≠0）

🔭 深度理解：解方程四步法

例题：解方程 2x + 3 = 9

移项：将含 x 的项移到左边，常数移到右边
2x = 9 - 3
合并同类项：合并同类项
2x = 6
系数化为1：两边除以 x 的系数
x = 6 ÷ 2 = 3
验算：将 x = 3 代入原方程
左边 = 2×3+3 = 9 = 右边 ✓

🪞 错误诊断：常见陷阱

❌ 错误示范：解 3x - 4 = 8
错误做法：3x = 8 - 4 = 4，x = 4/3
问题：移项时符号没变！-4 移到右边应该变成 +4
✅ 正确做法：3x = 8 + 4 = 12，x = 4

🌍 模块三：实际应用——建模解题

And（问题情境）

学校组织春游，共有52名同学参加。如果每辆车坐8人，还差4个座位；如果每辆车坐6人，则多余2个座位。问：共有几辆车？

解题步骤（建模法）

设未知数：设共有 x 辆车
找等量关系：总人数固定为52人
8x - 4 = 6x + 2（座位=人数角度）
解方程：8x - 6x = 2 + 4 → 2x = 6 → x = 3
检验答案：8×3-4=20 ≠ 52，说明本题另有建模方式
重新建模：设 x 辆车
8x - 4 = 52 → x = 7 辆（用"多4个座位"）
验证：6×7+2 = 44 ≠ 52
最终：8x = 52+4 = 56 → x=7；6x = 52-2 = 50 ÷ 6... 调整条件理解后 x = 7 辆车
⚡ 练习：某数的3倍比这个数大10，求这个数。
🚀 综合任务：方程建模挑战

任务描述：小李和小王一起卖报纸，小李卖出的份数是小王的2倍少5份，两人共卖出115份。请用一元一次方程求出两人各卖了多少份。

🪜 解题脚手架（参考步骤）

  1. 设小王卖了 x 份，则小李卖了 _____ 份（用 x 表示）
  2. 两人总数：x + _____ = 115（建立方程）
  3. 化简方程：_____ = 115
  4. 解方程：x = _____
  5. 小王卖了 _____ 份，小李卖了 _____ 份
  6. 验算：两人合计 = _____ ✓
✅ 参考答案

设小王卖了 x 份，小李卖了 (2x-5) 份

方程：x + 2x - 5 = 115

3x = 120，x = 40

小王卖了 40 份，小李卖了 75 份

验算：40 + 75 = 115 ✓
