📐 模块一：一次函数的图象

ABT 引入

已知（And）：你会在坐标系里描点、连线。

问题（But）：给你 y = 2x+1，每次都要算好几个点才能画——太慢了！

因此（Therefore）：掌握"截距法"，只需 2 个特殊点就能精准画出直线。

核心方法：截距法

令 x = 0，求纵截距：y = b → 点 (0, b)
令 y = 0，求横截距：x = −b/k → 点 (−b/k, 0)
连两点画直线，注意 k≠0

🎛️ 模块二：参数 k 和 b 的作用（互动探索）

拖动滑块，观察图象变化

2
1

归纳规律

当 k > 0 时

直线从左下到右上（单调递增）

当 k < 0 时

直线从左上到右下（单调递减）

b 决定纵截距

b > 0：直线与 y 轴正半轴交；b < 0：负半轴交

💡 模块三：待定系数法

例题：已知图象过 (1, 3) 和 (2, 5)，求解析式

设 y = kx + b
代入 (1,3)：k + b = 3  ···①
代入 (2,5)：2k + b = 5  ···②
② − ①：k = 2，回代得 b = 1
所以 y = 2x + 1 ✓
y = kx + b   →   两点代入 → 二元一次方程组 → 解出 k, b

⚠️ 典型易错点

易错 1
忘记 k ≠ 0 的条件
✅ 检查：若 k=0，y=b 是常数函数，不是一次函数
易错 2
混淆 k 和 b 的几何意义
✅ 记忆锚：k 控制"倾斜"（slope），b 控制"位置"（b=beginning）
易错 3
代点算错符号（负号漏掉）
✅ 代入后先算右边，再与左边对比确认等号成立

📌 本节小结

  • 一次函数 y = kx+b（k≠0）的图象是一条直线
  • k > 0：递增；k < 0：递减；|k| 越大，直线越"陡"
  • b 是纵截距，决定直线在 y 轴上的位置
  • 待定系数法：两点代入 → 二元一次方程组 → 求 k、b
