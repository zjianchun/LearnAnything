📖 模块一：有理数的概念与分类
📖 AND（背景）

你已经认识了自然数（1, 2, 3, …）和分数（½, ¾, …）。在生活中，我们还会遇到"零下5度"、"亏损3万元"……这些情况需要用负数来表示。

⚡ BUT（冲突）

问题来了：自然数、整数、分数、负数……这些数之间有什么联系？它们能统一在一个体系里吗？

💡 THEREFORE（解决）

数学家把所有"能写成两个整数之比"的数统一称为有理数（Rational Number，其中 rational 来自拉丁语 ratio = 比）。

📌 核心定义

凡是能写成 p/q（p、q 均为整数，且 q ≠ 0）形式的数，叫做有理数。

  • 整数可以看作分母为1的分数，如 3 = 3/1
  • 有限小数和循环小数都是有理数，如 0.5 = 1/2，0.333… = 1/3

🔬 放大镜——深度理解

注意：分类有两种角度！

角度一（按整数/分数）：整数 | 分数
角度二（按正负）：正有理数 | 0 | 负有理数
这两种分类方式都正确，互相补充！

📖 模块二：数轴——有理数的"地图"
📖 AND（背景）

有了数，我们需要一种方式来直观展示它们的位置关系。温度计、尺子……这些工具都隐含着一种数的排列方式。

💡 THEREFORE（解决）

数轴：一条带有刻度、方向和原点的直线，让每个有理数都有"家"。

📌 数轴三要素

  • 原点：表示0的点
  • 正方向：向右为正方向（箭头方向）
  • 单位长度：相邻整数间的距离相等

🪞 对比镜——正负对称

在数轴上，关于原点对称的两个点表示的数互为相反数。

+3 和 -3 互为相反数（距离原点相同，方向相反）
a 的相反数记作 -a（当 a = -3 时，-a = 3）

📌 绝对值

数轴上，一个数对应的点到原点的距离，叫做该数的绝对值，记作 |a|。

  • |3| = 3，|-3| = 3
  • |0| = 0
  • 正数的绝对值 = 本身；负数的绝对值 = 其相反数；0的绝对值 = 0
+

正数都大于0
−
负数在原点左边
负数都小于0
正数 > 0 > 负数  |  数轴上越靠右的数越大

📖 模块三：绝对值的深入理解
⚡ BUT（冲突）

很多同学会混淆：-(-3) = ?，|-3| = ?，这两个运算一样吗？

📌 绝对值的计算规则

  • 当 a > 0 时，|a| = a
  • 当 a = 0 时，|a| = 0
  • 当 a < 0 时，|a| = -a（变为正数）
⚠️ 反例镜——常见错误

错误：|−a| = a（这不一定对！）

若 a = -3，则 −a = −(−3) = 3，|−a| = |3| = 3 ✓
若 a = 5，则 −a = −5，|−a| = |−5| = 5 ✓
结论：|−a| = |a| 才是正确的！
🌍 情境镜——生活中的绝对值

两个城市之间的距离是绝对值的概念：A在原点，B在-5处，则A到B的距离是 |−5| = 5（公里），距离不分正负！
