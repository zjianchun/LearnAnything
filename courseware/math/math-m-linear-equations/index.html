📖 模块一：认识二元一次方程组

And（背景故事）

🛒 小红去文具店买钢笔和铅笔。她买了2支钢笔和3支铅笔共花了13元，买了1支钢笔和2支铅笔共花了8元。钢笔和铅笔各多少钱一支？

But（问题转化）

这里有两个未知数（钢笔价格 x，铅笔价格 y），同时有两个等量关系，需要联立求解！

方程①：2x + 3y = 13
方程②：x + 2y = 8

两个方程联合在一起，就是二元一次方程组！

Therefore（概念定义）

二元一次方程组：含有两个未知数，每个方程都是一次方程，这样的一组方程叫做二元一次方程组。

特征：① 两个方程 ② 两个未知数 ③ 每个方程最高次数为1 ④ 需要找同时满足两个方程的解

🔭 深度理解：解的概念

使两个方程都成立的一组 x, y 的值，叫做方程组的解。

验证 x=3, y=... 是否是解：

代入①：2×3 + 3y = 13 → y = 7/3（整数？）

代入①：2×1 + 3×... 试试 x=1, y=1：2+3=5 ≠ 13

正确答案：x=2, y=3（待解方程组后验证）

⚡ 判断：{ x+y=5, x-y=1 } 的解是 x=3, y=2 吗？
🔄 模块二：代入消元法

And（思路来源）

如果我们能把两个未知数的方程变成一个未知数，就能用已学的方法解了！

Therefore（代入法步骤）

例题：解方程组 { 2x + y = 7 ① , x - y = 2 ② }

用一个方程表示一个未知数
由②得：x = y + 2（选系数为1的项）
代入另一个方程
将 x = y+2 代入①：2(y+2) + y = 7
2y + 4 + y = 7
解一元一次方程
3y = 3，y = 1
回代求另一个未知数
x = y + 2 = 1 + 2 = x = 3
验算
①：2×3+1=7 ✓；②：3-1=2 ✓
解为 x=3, y=1

🪞 错误诊断

常见错误：代入时忘记乘以括号

❌ 2(y+2) + y = 7 → 2y+2+y=7（忘记2×2）

✅ 2(y+2) + y = 7 → 2y+4+y=7

⚡ 用代入法解：{ y = 2x ① , x + y = 9 ② }，x = ？
⚔️ 模块三：加减消元法

And（新思路）

当两个方程某个未知数的系数相同（或相反）时，直接相加或相减可以消去这个未知数！

Therefore（加减法步骤）

例题：{ 3x + 2y = 12 ① , x + 2y = 6 ② }

观察系数：y 的系数都是 2，可以用①-②消去 y
相减消元：(3x+2y)-(x+2y) = 12-6
2x = 6 → x = 3
代回求 y：代入②：3+2y=6 → 2y=3 → y=1.5
验算：①：9+3=12 ✓；②：3+3=6 ✓

🔭 方法选择指南

情况 推荐方法
某方程有 x=... 或 y=... 的形式 代入法
某未知数系数相同或互为相反数 加减法
系数不相等，需要配系数 先化系数再加减
⚡ 用加减法解：{ 2x + y = 8 ① , 2x - y = 4 ② }，y = ？
🚀 综合任务：鸡兔同笼

经典问题：鸡兔同笼，共有头30个，脚88只。问鸡、兔各有多少只？

🪜 解题脚手架

  1. 设鸡有 x 只，兔有 y 只
  2. 头的方程：x + y = ___
  3. 脚的方程：2x + ___y = 88（鸡2只脚，兔4只脚）
  4. 选择方法：_____（代入法 or 加减法）
  5. 解得 x = ___，y = ___
  6. 验算：头 = ___，脚 = ___ ✓
✅ 参考答案

设鸡 x 只，兔 y 只

{ x + y = 30 ① , 2x + 4y = 88 ② }

由①得 x = 30-y，代入②：2(30-y)+4y=88

60 - 2y + 4y = 88 → 2y = 28 → y = 14

x = 30 - 14 = 16

鸡16只，兔14只。验算：16+14=30 ✓，32+56=88 ✓
