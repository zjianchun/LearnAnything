📖 模块一：合并同类项
📖 AND（背景）

在算术中，2苹果 + 3苹果 = 5苹果，但 2苹果 + 3橙子不能合并成5个同类水果。代数中的"同类项"就是这个道理。

⚡ BUT（冲突）

当代数式中有很多项时，如何快速化简？比如 3x² + 2x − x² + 5x − 1 该如何处理？

💡 THEREFORE（解决）

把同类项的系数相加（利用分配律），这就是合并同类项。

📌 合并同类项法则

合并同类项：把系数相加，字母和指数不变。

  • 依据：分配律 ma + na = (m+n)a
  • 例：3x + 5x = (3+5)x = 8x
  • 例：3x² - x² = (3-1)x² = 2x²
化简：3x² + 2x − x² + 5x − 1
↓ 找同类项（颜色相同的是同类项）
3x² 和 -x² 是同类项， 2x 和 5x 是同类项， -1 单独一项
↓ 合并
(3-1)x² = 2x² (2+5)x = 7x -1
↓ 结果
2x² + 7x − 1
⚠️ 反例镜——不能合并的情况

只有同类项才能合并，不同类的不能合并！

✗ 错误：3x + 2 = 5x（x和常数不同类）
✗ 错误：3x² + 2x = 5x³（合并时字母和次数不变！）
✓ 正确：3x² + 2x² = 5x²（系数相加，x²不变）

📖 模块二：去括号法则
⚡ BUT（冲突）

在整式加减中，经常遇到括号。括号前是正号还是负号，处理方式完全不同，稍不注意就会出错！

📌 去括号法则

  1. 括号前是 + 号：去掉括号，括号内各项符号不变
  2. 括号前是 − 号：去掉括号，括号内各项符号全部变号
📐 例1：正号前的括号
+(3x − 2y + 1)
= +3x − 2y + 1（符号不变）
= 3x − 2y + 1
📐 例2：负号前的括号
−(3x − 2y + 1)
= −3x + 2y − 1（每项都变号！）
= -3x + 2y - 1
🔬 放大镜——为什么要变号？

本质是分配律：−(3x − 2y + 1) = (−1)×(3x − 2y + 1)

= (−1)×3x + (−1)×(−2y) + (−1)×1
= −3x + 2y − 1
记忆口诀："负号进括号，括号内变天地"
去括号并化简：5x − (3x − 2) + 4 = ?
📖 模块三：整式加减综合
💡 THEREFORE（解决）

整式加减的完整流程：①去括号 → ②找同类项 → ③合并同类项 → ④化简结果

1

去括号

处理括号，正号不变号，负号全变号

2

找同类项

用下划线或括号标记出同类项（可用不同颜色区分）

3

合并同类项

系数相加，字母和指数不变

4

写最简结果

按次数从高到低排列各项

📐 完整例题：(2x²-3x+1)-(x²+2x-5)
= 2x² - 3x + 1 - x² - 2x + 5（去括号，负号变号）
= (2x²-x²) + (-3x-2x) + (1+5)（找同类项）
= x² - 5x + 6（合并同类项）
🌍 情境镜——用整式解决实际问题

一个矩形花圃，长为(2x+3)米，宽为(x-1)米。另一个正方形花圃，边长为(x+2)米。两者周长之差是多少？

矩形周长 = 2(2x+3+x-1) = 2(3x+2) = 6x+4
正方形周长 = 4(x+2) = 4x+8
差 = (6x+4)-(4x+8) = 6x+4-4x-8 = 2x-4（米）
化简：3(a+2b) - 2(2a-b) = ?
