📖 模块一：一元一次不等式的概念

And（对比方程）

你已经学会解方程 2x + 3 = 7，找到唯一的 x 值。但是，如果条件是 "2x + 3 > 7" 呢？x 不是唯一一个值，而是一大片！

But（方程 vs 不等式）

⚖️ 方程

2x + 3 = 7

解：x = 2（唯一值）

数轴上：一个点

📏 不等式

2x + 3 > 7

解：x > 2（无穷多）

数轴上：一条射线

Therefore（定义）

一元一次不等式：只含一个未知数，未知数最高次数为1，且含有不等号的不等式。

标准形式：ax + b > c（或 <、≤、≥，其中 a ≠ 0）

其解集是满足不等式的所有 x 的值的集合，通常是一条射线（无穷多个解）。

🔭 数轴表示法（可视化镜）

x > 2 ← ●○——————————→ 空心圆，向右箭头
x ≥ 2 ← ●●——————————→ 实心圆，向右箭头
x < 2 ←——————————○●→ 空心圆，向左箭头
x ≤ 2 ←——————————●●→ 实心圆，向左箭头

口诀：有等号（≤或≥）→ 实心圆；无等号（<或>）→ 空心圆

⚡ 练习：x ≤ -1 在数轴上，端点 -1 应该用什么圆？箭头向哪边？
🧮 模块二：解一元一次不等式（步骤详解）

And（类比方程的解法）

解不等式的步骤和解方程几乎完全一样——移项、合并、系数化1。唯一的区别是：乘除负数时不等号翻转！

Therefore（完整解题流程）

例1（系数为正）：解 2x - 4 > 6

移项：2x > 6 + 4 → 2x > 10
系数化1（除以正数2，不翻转）：x > 5
数轴表示：5右侧，空心圆，向右箭头

例2（系数为负）：解 -3x + 6 > 15

移项：-3x > 15 - 6 → -3x > 9
除以 -3（负数！翻转！）：x < -3
数轴表示：-3左侧，空心圆，向左箭头

🔭 错误防雷镜

⚠️ 最常见错误类型：

错误做法原因正确做法
-2x > 8 → x > -4忘记翻转x < -4 ✓
x-3>5 → x>2移项符号错x > 8 ✓
2x>-6 → x>-6漏除以2x > -3 ✓

🌍 模块三：一元一次不等式应用

And（应用题类型）

不等式应用题的关键是找到"不等量关系"，常见于：最多/最少问题、满足条件问题、行程问题等。

Therefore（解题示例）

例：某工厂计划生产零件，第一天生产了200个，此后每天生产150个。问至少需要几天才能完成1000个的订单？

设未知数：设还需要 x 天（不含第一天）
建立不等式：200 + 150x ≥ 1000
解不等式：150x ≥ 800 → x ≥ 800÷150 ≈ 5.33
取整数：x 至少为 6（天数必须是整数）
解答：共需 1+6 = 7 天（含第一天）

🔭 注意：整数约束

在实际应用中，x 常常要求是整数（天数、人数、个数等）。

若解出 x > 5.33，且 x 为正整数，则 x ≥ 6（向上取整）

若解出 x < 7.8，且 x 为正整数，则 x ≤ 7（向下取整）

🚀 综合任务：打折购物问题

任务：某商场打折促销，衬衫原价每件80元，裤子原价每件120元。小张的购物预算不超过500元，他想买2件衬衫，其余预算全部买裤子。最多能买几条裤子？

🪜 解题脚手架

  1. 设买 x 条裤子
  2. 衬衫费用：2 × 80 = _____ 元
  3. 裤子费用：120 × x 元
  4. 预算不超过500：_____ + 120x ≤ 500
  5. 解不等式：120x ≤ _____ → x ≤ _____
  6. 最多买 _____ 条裤子
✅ 参考答案

160 + 120x ≤ 500

120x ≤ 340

x ≤ 2.83

x 最大整数值为 2，最多买 2 条裤子。

验算：160 + 240 = 400 ≤ 500 ✓
