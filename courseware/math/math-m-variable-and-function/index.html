📖 模块 1：变量与常量

🌡️ 气象观测员

And 你已经会用数字表示生活中的量：温度、距离、价格……

But 有些量会随时间变化（今天 22℃，明天 18℃），有些量却始终不变（1 天有 24 小时，圆周率 π 永远是 3.14...）。

Therefore 数学家把会变的量叫做变量，把不变的量叫做常量。认识这个区别，是理解函数的第一步。

📝 核心概念

1
常量（Constant）：在某个研究过程中，数值始终不变的量。
例：π ≈ 3.14，光速 ≈ 3×10⁸ m/s，一周有 7 天。
2
变量（Variable）：在某个研究过程中，数值可以变化的量。
例：气温、汽车速度、商品价格、弹簧长度。
3
关键：常量与变量的区分是相对的——在不同的研究情境下，同一个量可能是常量，也可能是变量。

🤖 函数就像自动售货机

请选择一个 x 值，体验函数 y = 2x + 1

💡 关键：每个 x 值只能对应唯一一个 y 值，这就是函数！

📝 函数的定义

若变量 y 随变量 x 的变化而变化，
且对于 x 的每一个值，y 都有唯一确定的值与之对应，
则称 y 是 x 的函数。
①
自变量：可以自由取值的变量（x），像"输入"
②
因变量：随自变量变化的变量（y），像"输出"
③
唯一性：x 的每个值，y 只有且只有一个对应值——这是成为函数的必要条件
🏧
函数 = 自动售货机
投入 x（硬币）→ 吐出唯一的 y（饮料）
同样的硬币永远出同一种饮料 = 唯一对应

📊 模块 3：函数的三种表示方法

同一个函数可以用三种不同方式来表达——就像同一件事可以用文字、表格或图片来描述：

y = 2x + 1

解析式法：用数学表达式直接写出 y 与 x 的关系。

  • 优点：简洁，可以计算任意 x 对应的 y
  • 适用：关系明确、可用公式表达的函数
  • 例：路程 s = 60t（以 60km/h 匀速行驶 t 小时）
x（自变量）-2-10123
y = 2x+1（因变量）-3-11357

列表法：用表格列出自变量与因变量的对应值。

  • 优点：直观，易于读取具体数值
  • 适用：数据有限、规律不明显的函数（如实验数据）

y = 2x + 1 的图像（直线）

图像法：用坐标平面上的图形表示函数关系。

  • 优点：直观展示函数的变化趋势
  • 适用：需要观察整体趋势、分析极值等情况

⚠️ 高频易错点

  • 易错1：误认为"有对应关系就是函数"——关键要看 x 的每个值对应的 y 是否唯一
  • 易错2：混淆自变量和因变量——"y 是 x 的函数"中，x 是自变量，y 是因变量
  • 易错3：三种表示法中，列表法只能展示有限个对应值，不代表函数只在这些点有定义
  • 易错4：解析式 y = x² 中，x 可以取任意实数；但 y = √x 中，x 必须 ≥ 0（注意定义域）

🧠 深层理解

🔍 镜头一：函数与"唯一性"的哲学

为什么函数要求"唯一对应"？因为函数是一种确定性规则——给定输入，输出结果可预测。这就像：每次你按同一个电梯按钮，电梯不会随机去不同楼层。数学中的函数是自然界"因果关系"的抽象。

🔍 镜头二：函数不等于公式

很多学生以为函数一定是 y = ... 这样的代数式。其实不然：

  • 气象局记录的"日期→最高气温"表格，是函数（无公式）
  • 你的"姓名→学号"对应关系，是函数（非数值）
  • 一个圆 x²+y²=1 中，y 不是 x 的函数（一个 x 对应两个 y）

函数的本质是对应规则，不一定是公式。

🔍 镜头三：自变量与因变量的"视角"问题

同一个关系 y = 2x + 1，可以说"y 是 x 的函数"，也可以反过来说"x = (y-1)/2，x 是 y 的函数"。改变研究视角，自变量和因变量会互换。数学的美丽在于：你选择关注哪个方向的变化。

💭 你的理解：

用自己的话解释：为什么 x²+y²=4（圆）不是函数，而 y=x²（抛物线）是函数？

圆方程中，当 x=0 时，y=2 和 y=-2 都满足，一个 x 对应两个 y，违反唯一性→不是函数。而 y=x² 中，每个 x 都只对应一个 y（例如 x=2 时只有 y=4），满足唯一性→是函数。

📌 课程小结

🗺️ 本课知识地图

1
常量 vs 变量：研究过程中不变的是常量，变化的是变量
2
函数定义：y 是 x 的函数 ⟺ 每个 x 对应唯一 y
3
三种表示：解析式（最简洁）、列表（最直观）、图像（最形象）
4
判断函数：核心问题——"一个 x 能不能对应多个 y？"
🏧
终极记忆：函数 = 自动售货机规则
x = 你投入的硬币（自变量）
y = 机器吐出的饮料（因变量）
同一枚硬币 → 永远出同一种饮料（唯一对应）
这个规则本身 = 函数
