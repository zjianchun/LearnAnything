📖 模块一：相交线产生的角
🌟 ABT引入：
And：两条路在十字路口相交，形成4个角。
But：这4个角并不是随便的——它们之间有特殊的关系！
Therefore：数学家发现：相交的两直线形成的4个角中，对顶角相等，邻补角互补！

🔴 对顶角

两直线相交，位置相对的两个角。
性质：对顶角相等（∠1=∠3，∠2=∠4）

🟡 邻补角

相邻的两个角（共用一条边）。
性质：邻补角互补（相加=180°）

🏋️ 即时练习：若一个角为 70°，则它的邻补角为？

✂️ 模块二：截线产生的角（同位角、内错角、同旁内角）
🔑 概念：一条直线（截线）截两条平行线，会产生8个角。这8个角中，有3种特殊的角对：

同位角

在截线同侧，位置相同（如都在截线左上方）

内错角

在两线之间，截线两侧（形状像"Z"）

同旁内角

在两线之间，截线同侧（形状像"U"）

🏋️ 即时练习：内错角位于截线的哪个位置？

∥ 模块三：平行线的判定与性质
🔑 判定 vs 性质的区别：
• 判定：已知角度关系 → 推出两线平行
• 性质：已知两线平行 → 推出角度关系
这是逻辑的"双向通道"！

📌 平行线的判定（3个条件，满足任一即可）

① 同位角相等 → 两直线平行（简写：同位角相等，两直线平行）

② 内错角相等 → 两直线平行

③ 同旁内角互补（和为180°）→ 两直线平行

📌 平行线的性质（两线平行时成立）

① 两直线平行 → 同位角相等

② 两直线平行 → 内错角相等

③ 两直线平行 → 同旁内角互补

🏋️ 即时练习：已知 a∥b，同位角 ∠1 = 65°，则 ∠2 = ？（∠1 和 ∠2 是同位角）

🎯 综合任务：求角度链

已知 a∥b，截线 c 与 a、b 相交，形成角 ∠1=50°（∠1 在直线 a 上方，截线 c 右侧）。

🪜 解题脚手架

  1. 画出图形，标注 ∠1=50°
  2. 求 ∠1 的对顶角（在 a 下方，截线左侧）= ？
  3. 求与 ∠1 同位的角（在 b 上方，截线右侧）= ？（用平行线性质）
  4. 求 ∠1 的同旁内角 = ？（与∠1在截线同侧，在两线之间）
  5. 求 ∠1 的内错角 = ？
