📖 模块一：无理数的发现
📖 AND（背景）

古希腊数学家毕达哥拉斯相信"万物皆有理数"。他们认为，任何两段线的长度之比都可以用整数之比表示。

⚡ BUT（冲突）

一个正方形，边长为1，对角线是多少？用勾股定理：对角线² = 1² + 1² = 2，所以对角线 = √2。但是 √2 = ? 它不等于任何分数！

💡 THEREFORE（解决）

√2 是一种新类型的数——无理数。它的小数展开是无限且不循环的：√2 ≈ 1.41421356…

📌 有理数与无理数的本质区别

有理数：可以写成分数 p/q（整数比）→ 小数展开是有限小数或无限循环小数

无理数：不能写成分数形式 → 小数展开是无限不循环小数

📖 模块二：实数体系
💡 THEREFORE（解决）

有理数和无理数合在一起，构成了实数（Real Numbers，ℝ）。实数对应数轴上的每一个点，与数轴上的点一一对应。

📌 实数与数轴

  • 实数与数轴上的点一一对应
  • 数轴上每一个点都对应一个实数
  • 每个实数都对应数轴上唯一一个点
  • 有理数在数轴上"稠密"，但无理数填满了缝隙
🌍 情境镜——生活中的无理数

圆的周长 C = 2πr（含π，无理数）；正方形对角线 = 边长 × √2；黄金比例 ≈ 1.618（无理数）；音乐中的十二平均律使用了 2的12次方根（无理数）

📖 模块三：实数的运算与大小比较
💡 THEREFORE（解决）

实数的运算法则和有理数相同（加减乘除），但比较大小时，对于含根号的数，需要用估算或化简的方法。

📌 平方根和算术平方根

  • 算术平方根：非负数a的非负平方根，记作 √a（a ≥ 0）
  • 平方根：a²= b，则a是b的平方根（有±两个，b > 0时）
  • √a × √a = a（a ≥ 0）
  • √(a²) = |a|（注意绝对值！）
⚠️ 反例镜——√(a²)的陷阱

√(a²) ≠ a！正确结论是 √(a²) = |a|

当a = 3时：√(3²) = √9 = 3 = |3| ✓
当a = -3时：√((-3)²) = √9 = 3 = |-3| ✓
不能写成√((-3)²) = -3（错误！√结果必须非负）

📌 实数大小比较

  • 利用估算：√3 ≈ 1.732，则 √3 与 1.8 比较：1.732 < 1.8，所以 √3 < 1.8
  • 两边平方（两数都是正数时）：比较 √5 和 2，都是正数，5 vs 4，5>4，所以 √5 > 2
