📖 模块一：认识不等式

And（生活场景）

🏪 超市规定：购物满100元才能享受折扣。小明的购物车总价为 x 元，享受折扣的条件是什么？

But（数学表达）

这个条件不是一个等式，而是一种"大于等于"的关系：

x ≥ 100

这就是一个不等式——用不等号连接两个代数式的式子。

Therefore（不等号符号表）

符号 读法 数轴表示 端点
<小于向左的箭头空心圆（不含）
>大于向右的箭头空心圆（不含）
≤小于等于向左的箭头实心圆（含）
≥大于等于向右的箭头实心圆（含）

⚙️ 模块二：不等式的性质（⚠️ 重要！）

And（等式 vs 不等式）

等式两边加减任意数、乘除任意非零数，等号不变。不等式呢？大部分情况相同，但有一个惊天大坑！

Therefore（三大性质）

性质1：加减不变

若 a > b，则 a + c > b + c（c 为任意实数）

例：3 > 1，两边+5：8 > 6 ✓

性质2：乘除正数不变

若 a > b 且 c > 0，则 ac > bc

例：3 > 1，两边×2：6 > 2 ✓

⚠️ 性质3：乘除负数要翻转！！！

若 a > b 且 c < 0，则 ac < bc（不等号方向改变！）

例：3 > 1，两边×(-2)：-6 < -2 ✓（符号翻转）

记住：乘/除负数，不等号翻个身！

🪞 错误诊断

❌ 经典错误：解不等式 -2x > 6

错误：x > 6÷(-2) = -3（忘记翻转）

✗ 错误！两边除以 -2 需要翻转不等号

✅ 正确：-2x > 6 → x < -3（翻转！）

🔗 模块三：不等式组

And（问题场景）

🎒 小明要参加活动，行程时间需要满足：超过1小时（路上堵车需要时间准备），但不超过3小时（不能耽误太久）。如何表示？

Therefore（不等式组解法）

同时满足两个不等式：{ x > 1 且 x ≤ 3 } ，解集为 1 < x ≤ 3

例题：解不等式组 { 2x - 1 > 3 ① , x + 2 < 6 ② }

分别解每个不等式
①：2x > 4 → x > 2
②：x < 4 → x < 4
在数轴上画出各解集
x > 2：在2右侧（空心）；x < 4：在4左侧（空心）
取公共部分（交集）
公共解集：2 < x < 4

记忆口诀：

同向取中间（都向右/都向左→中间那段）

反向取两端（一左一右→两端各一段，可能无解）

🚀 综合任务：生活中的不等式

任务：某同学前三次数学考试成绩为78分、85分、91分。他希望四次测试的平均分不低于85分。第四次至少要考多少分？

🪜 解题脚手架

  1. 设第四次考试得 x 分
  2. 四次平均分 = (78+85+91+x) ÷ 4
  3. 条件：平均分 ≥ 85，即：(78+85+91+x) ÷ 4 _____ 85
  4. 化简：____ + x ≥ ____
  5. x ≥ ____
  6. 答：第四次至少要考 ____ 分
✅ 参考答案

(78+85+91+x)/4 ≥ 85

254 + x ≥ 340

x ≥ 86

第四次至少要考86分。
