🔬 神经元的基本结构

神经元 = 细胞体 + 突起（树突 + 轴突）
神经元是高度特化的细胞，能够接受刺激、产生兴奋并传导神经冲动。
结构特征功能
细胞体含细胞核，体积较大代谢中心，整合信息
树突短而多分支接受其他神经元的信号
轴突长，末端分支（轴突末梢）将信号传向其他细胞
髓鞘包裹轴突的绝缘层加速神经冲动传导
突触神经元之间的连接处化学信号传递的关键
💡 突触的工作原理：
突触前膜释放神经递质 → 递质穿越突触间隙 → 与突触后膜受体结合 → 产生电信号。
注意：神经冲动在突触处只能单向传递（突触前→突触后）！

🧠 中枢神经系统

脑 + 脊髓
位于颅骨和脊椎骨内，受到骨骼保护
  • 大脑：思维、语言、记忆、随意运动
  • 小脑：协调运动、维持身体平衡
  • 脑干：调节呼吸、心跳等基本生命活动
  • 脊髓：传导信息 + 控制低级反射

🔗 周围神经系统

脑神经 + 脊神经
连接中枢与身体各器官
  • 脑神经（12对）：连接脑与头面部
  • 脊神经（31对）：连接脊髓与躯干四肢
  • 传入神经：将信号传向中枢
  • 传出神经：将指令传向效应器
🔴 脑的高级功能：大脑皮层是思维活动的物质基础，具有语言中枢（人类特有）、视觉中枢、听觉中枢、躯体运动中枢和躯体感觉中枢等功能区。

条件反射与非条件反射

根据获得方式和神经联系的不同，反射可分为两大类

⚡ 非条件反射

生来就有，不需要学习，由低级中枢控制
  • 遗传决定，永久保持
  • 弧数固定，反应相同
  • 中枢在脊髓或脑干
  • 种内各个体相同
例子：手被烫缩回（缩手反射）、膝跳反射、排尿反射、呼吸反射、婴儿吮吸

🧠 条件反射

后天获得，需要训练，由大脑皮层控制
  • 后天学习获得
  • 需要强化才能维持
  • 中枢在大脑皮层
  • 个体间有差异
例子：巴甫洛夫实验（铃声→分泌唾液）、看到梅子流口水、红灯停绿灯行

📊 对比表格

比较项目非条件反射条件反射
形成方式先天遗传后天训练
神经联系固定不变暂时联系
消退性不会消退不强化则消退
控制中枢脊髓/脑干大脑皮层
适应意义维持基本生命活动适应复杂变化环境
🌟 人类特有的条件反射：人类不仅能对具体刺激（铃声、光）建立条件反射，还能对语言和文字建立条件反射——这是人类思维的基础，也是人与动物的本质区别之一。

第1题（单选）：关于神经元的结构，下列说法正确的是？
A 每个神经元只有一个树突和一个轴突
B 树突通常短而多分支，轴突通常细长
C 神经冲动可以双向通过突触传递
D 神经元没有细胞核
错因分析：A项错误——树突多分支、轴突只有一条；C项错误——突触传递具有单向性，只能由突触前膜到突触后膜；D项错误——神经元有细胞核，是正常的有核细胞。B项正确描述了树突（多、短、分支多）和轴突（细长）的形态特征。
第2题（单选）：完成缩手反射时，神经冲动传导的正确顺序是？
A 效应器→传出神经→神经中枢→传入神经→感受器
B 感受器→神经中枢→传入神经→传出神经→效应器
C 感受器→传入神经→神经中枢→传出神经→效应器
D 感受器→传出神经→神经中枢→传入神经→效应器
错因分析：反射弧的传导方向必须记清楚：感受器接受刺激后，冲动经传入神经传到神经中枢（这是"传入"——向内传），神经中枢分析后发出指令经传出神经传到效应器（这是"传出"——向外传）。顺序是固定的：感受器→传入神经→神经中枢→传出神经→效应器。
第3题（单选）：下列反射中，属于条件反射的是？
A 婴儿出生后立即会吮吸乳汁
B 医生用小锤叩击膝盖产生膝跳反射
C 手碰到烫水时迅速缩手
D 看到酸梅图片时口腔分泌唾液
错因分析：A（吮吸）、B（膝跳）、C（缩手）都是先天就有、不需要学习的非条件反射，由遗传决定，由脊髓或脑干控制。D（看图流口水）需要通过生活经验学习获得，是语言文字刺激引起的条件反射，由大脑皮层控制，是人类高级神经活动的体现。
