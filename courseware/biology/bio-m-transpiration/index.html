蒸腾作用

一棵大树每天能蒸腾数百升水，这不是浪费，而是植物精巧的水分运输动力系统。

And — 已知

根从土壤吸收水分，水通过导管向上运输到叶片。叶片上有气孔，可以和外界进行气体交换。

But — 问题

但水分从根运到十几米高的树冠，靠什么驱动力？为何炎热天气移植树木要去除大量叶片？

Therefore — 探究

因此我们学习蒸腾作用——水分从叶片气孔散失的过程，理解它如何驱动水分在植物体内运输。

📌 核心目标：描述蒸腾作用的全程路径；分析影响蒸腾速率的因素；解释蒸腾作用对植物和环境的意义。

预备知识

🔍 知识预检

回顾已学内容：

✅ 气孔由两个保卫细胞围成，主要在下表皮
✅ 水分从根毛→皮层→导管→向上运输
❌ 蒸腾作用对植物毫无益处，只是水分流失
✅ 气温高、风速大、湿度低时蒸腾速率加快
模块一

💧 蒸腾作用的全程路径

💧 完整路径：土壤水分 → 根毛细胞（渗透吸水）→ 皮层细胞 → 木质部导管 → 茎导管 → 叶片导管 → 叶肉细胞间隙 → 气孔下腔 → 气孔 → 大气（以水蒸气形式散失）

蒸腾作用的三大功能

🚀
驱动水分运输

蒸腾拉力是水分从根向上运输到高处的主要动力

🌡️
调节体温

水分蒸发吸热，降低叶片温度，防止被灼伤

🌿
促进无机盐运输

水分运输的同时，携带溶解在水中的无机盐到达各组织

🖱️ 拖拽练习：将水分运输的路径正确排序

  • 🔀茎的导管向上运输
  • 🔀土壤中的水分
  • 🔀水蒸气从气孔散失到大气
  • 🔀根毛细胞渗透吸水
  • 🔀叶片细胞间隙→气孔下腔

影响因素对蒸腾速率的影响原因
气温升高蒸腾速率增大水分子运动加快，蒸发加速

🌳
移植树木为何要剪去大量枝叶？
移植时根系受损，吸水能力大幅下降。若不剪枝，大量叶片继续蒸腾，失水多于吸水，植物会因失水而死亡。去除叶片可以减少蒸腾，有助于成活。
模块三

🌍 蒸腾作用的生态意义

🚿

水循环的驱动力

植物蒸腾是水循环的重要环节，大片森林的蒸腾作用可以影响局部气候，增加降水。

🌡️

调节小气候

树木蒸腾吸收大量热量，使林区气温较低、空气湿润。城市绿化树木可降低城市热岛效应。

🧪

促进无机盐吸收利用

蒸腾作用产生的拉力促使根吸水，水携带无机盐通过导管向上运输，供各器官利用。

🔑 记忆要点：蒸腾作用 = 水分散失 + 降温 + 驱动水运输。植物每天有90%以上吸收的水分通过蒸腾散失，只有不到10%用于光合等生命活动。

Q1. 蒸腾作用是指植物体内的水分以______形式，通过______散失到大气中的过程。
❌ 错因分析：蒸腾作用是水分从液态转变为水蒸气，通过叶片的气孔散失到大气中的过程。根毛是吸水的，导管是运水的，不是蒸腾作用发生的部位。
Q2. 炎热夏天，在烈日下大树周围的温度比空旷地带低，主要原因是？
❌ 错因分析：蒸腾作用中水分由液态变为气态，这个相变过程会吸收大量热量，从而使周围气温降低。这是蒸腾作用调节体温和小气候的重要功能。遮阴只是次要原因。
Q3. 下列条件中，会使蒸腾速率明显减慢的是？
❌ 错因分析：蒸腾速率取决于叶片与大气之间的水蒸气浓度差。当空气湿度升高（如雨天），大气中水蒸气浓度接近叶片气腔内的浓度，浓度差减小，蒸腾速率显著降低。高温、强光、大风都会加快蒸腾。

本节重点回顾

  • 蒸腾路径：土壤→根毛→皮层→导管→茎导管→叶导管→气腔→气孔→大气
  • 蒸腾拉力是水分向上运输的主要动力
  • 温度高、光照强、风速大 → 蒸腾加快；湿度高 → 蒸腾减慢
  • 蒸腾意义：驱动水分运输、调节体温、参与水循环调节小气候
  • 移植树木剪叶：减少蒸腾→减少失水→提高成活率
