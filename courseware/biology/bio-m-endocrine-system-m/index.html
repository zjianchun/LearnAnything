激素调节与内分泌系统

探索人体的"化学信使"——激素如何在血液中传递指令，精确调控生命活动

🫀 内分泌腺 ⚗️ 激素特性 🩸 血糖调节 🔄 拮抗协同
课程导入

用 ABT 叙事框架了解本节课的核心问题

我们已知道神经系统通过电信号快速协调身体反应。人体内还有另一套调节系统——内分泌系统，通过分泌化学物质（激素）来调节各器官的功能。
但激素与神经信号不同：它不走神经纤维，而是直接分泌到血液中，随血液循环到达全身。激素如何精确"找到"目标器官？血糖升高时为什么不会一直升下去？
因此，本节课我们将认识主要内分泌腺和它们分泌的激素，理解激素调节的特点，以及胰岛素和胰高血糖素如何协同调节血糖稳定。

📋 本节学习目标

  • 说明激素的概念及其特性（微量、高效、通过体液运输）
  • 识别主要内分泌腺：垂体、甲状腺、胰岛、肾上腺、性腺
  • 理解血糖调节中胰岛素与胰高血糖素的拮抗作用
  • 比较神经调节与激素调节的异同
  • 了解激素分泌异常导致的疾病（糖尿病、侏儒症等）

激素与体液调节

激素是内分泌腺分泌的化学物质，通过体液（血液）运输调节生命活动

👆 动画演示激素随血液循环到达靶器官的过程

⚗️ 激素的三大特性

① 微量高效
激素的量极少（ng/mL级别），但调节效果显著。一旦分泌过多或过少，都会引起严重的代谢紊乱。
② 通过体液运输
激素直接分泌到体液（血液）中，随血液循环运输到全身，不通过导管，因此内分泌腺也称"无管腺"。
③ 作用于靶器官/靶细胞
激素只对含有特定受体的细胞（靶细胞）起作用，具有高度的专一性。比如胰岛素主要作用于肝脏和肌肉细胞。
💡 与外分泌腺的区别
外分泌腺（如唾液腺）有导管，分泌物通过导管排出。内分泌腺无导管，分泌物（激素）直接进入血液。

血糖调节——拮抗与协同

正常血糖浓度约为 0.8~1.2 g/L，胰岛素与胰高血糖素相互拮抗，维持血糖稳定

👆 动画展示饭后/饥饿时血糖的调节过程

💉 胰岛素（胰岛B细胞分泌）

作用：降低血糖
  • 促进组织细胞摄取葡萄糖
  • 促进葡萄糖氧化分解
  • 促进葡萄糖合成糖原（肝糖原、肌糖原）
  • 抑制非糖物质转化为葡萄糖
⚠️ 胰岛素分泌不足或靶细胞不敏感 → 糖尿病（血糖长期偏高）

🧬 胰高血糖素（胰岛A细胞分泌）

作用：升高血糖
  • 促进肝糖原分解为葡萄糖
  • 促进非糖物质转化为葡萄糖
  • 抑制葡萄糖的氧化分解
  • 与胰岛素作用相反（拮抗）
💡 当血糖过低（低血糖）时胰高血糖素分泌增多，防止血糖继续下降
🔄 负反馈调节：血糖升高 → 胰岛素↑、胰高血糖素↓ → 血糖降低 → 胰岛素↓、胰高血糖素↑ → 血糖升高……如此循环，血糖维持在正常范围内。这是一种负反馈调节机制。

第1题（单选）：下列关于激素的说法，正确的是？
A 激素通过神经纤维传递到靶器官
B 激素量多，在体内起持久的调节作用
C 激素微量高效，通过体液（血液）运输，作用于靶细胞
D 所有细胞对激素的反应方式都相同
错因分析：A项错误——激素通过血液（体液）运输，不走神经纤维；B项错误——激素的特点是"微量"而非量多；D项错误——激素只作用于有特定受体的靶细胞，不同细胞对激素的反应不同。C项正确概括了激素的三大特性：微量高效、通过体液运输、作用于靶细胞。
第2题（单选）：饭后血糖浓度升高，此时机体的调节是？
A 胰岛A细胞分泌胰岛素增多，血糖降低
B 胰岛B细胞分泌胰岛素增多，促进组织细胞摄取和利用葡萄糖
C 胰岛B细胞分泌胰高血糖素增多，血糖继续升高
D 肾上腺分泌肾上腺素降低血糖
错因分析：关键要分清A细胞和B细胞：胰岛B细胞分泌胰岛素（降低血糖）；胰岛A细胞分泌胰高血糖素（升高血糖）。饭后血糖升高，应由B细胞增加胰岛素分泌来降低血糖。A项把细胞类型搞错了；C项不仅细胞类型错了，方向也错了；D项肾上腺素不直接参与血糖的精细调节。
第3题（单选）：幼年时期甲状腺激素分泌不足，会导致？
A 巨人症
B 侏儒症（但智力正常）
C 甲亢（甲状腺功能亢进）
D 呆小症（身材矮小且智力低下）
错因分析：各种激素异常的疾病要分清楚：A（巨人症）是幼年生长激素过多；B（侏儒症，智力正常）是幼年生长激素过少；C（甲亢）是甲状腺激素过多；D（呆小症）是幼年甲状腺激素不足——因为甲状腺激素不仅促进代谢，还促进神经系统的发育，所以缺乏会同时影响身材和智力。
