盐的性质

掌握盐与金属、酸、碱、盐之间的四类反应规律，理解复分解反应发生条件

学习目标

  • 理解盐的化学性质
  • 掌握盐与金属、酸、碱、盐的反应规律
  • 理解复分解反应发生条件

盐的化学性质

盐 + 金属 → 新盐 + 新金属

盐 + 酸 → 新盐 + 新酸

盐 + 碱 → 新盐 + 新碱

盐 + 盐 → 新盐 + 新盐

盐与金属反应

例如：Zn + CuSO₄ → ZnSO₄ + Cu

盐与酸反应

例如：Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂↑

盐与碱反应

例如：CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄

盐与盐反应

例如：AgNO₃ + NaCl → AgCl↓ + NaNO₃

复分解反应发生条件

生成物中必须有沉淀、气体或水

总结

1.
盐的化学性质包括与金属、酸、碱、盐的反应。
2.
复分解反应发生条件是生成物中必须有沉淀、气体或水。
3.
盐的化学反应在日常生活和工业生产中都有广泛应用。
