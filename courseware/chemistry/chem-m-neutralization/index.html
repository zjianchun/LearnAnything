Concept Building

💡 概念建构：什么叫中和反应？

中和反应定义

酸 + 碱 → 盐 + 水 的化学反应，叫做中和反应。

📌 中和反应的微观实质

H+ + OH- → H2O

酸提供的氢离子(H⁺)和碱提供的氢氧根离子(OH⁻)结合生成了水分子(H₂O)，这就是中和反应的微观本质。

📌 中和反应的化学方程式示例

HCl + NaOH → NaCl + H2O
盐酸 + 氢氧化钠 → 氯化钠 + 水
H2SO4 + 2NaOH → Na2SO4 + 2H2O
硫酸 + 氢氧化钠 → 硫酸钠 + 水

⚠️ Common Pitfalls

⚠️ 避坑指南：三个最常见的错误

❌ 坑1：把"生成盐和水"的都叫中和

纠正：必须是酸+碱生成盐+水，才叫中和。
CuO + 2HCl → CuCl₂ + H₂O 虽生成盐和水，但CuO是氧化物不是碱，不是中和反应。

❌ 坑2：忽略中和后溶液不一定中性

纠正：只有恰好完全反应时pH才≈7。酸过量时溶液仍呈酸性，碱过量时仍呈碱性。

❌ 坑3：忽略离子守恒

纠正：中和反应本质是H⁺ + OH⁻ → H₂O，Na⁺和Cl⁻等"旁观离子"没有消失，仍然留在溶液中形成盐。

Applications

🌍 中和反应在生活中的应用

学完概念，现在回来解答开头的问题：

🦟 蚊虫叮咬

现象：蚊子叮咬时会注入蚁酸（甲酸），让人皮肤发痒。

原理：肥皂水是碱性，涂在皮肤上可以中和蚁酸的酸性，止痒。

🏥 胃药

现象：胃酸过多会导致胃部不适。

原理：氢氧化铝是碱性药物，能中和多余的胃酸（盐酸）。

🏭 污水处理

现象：工厂排放的废水常呈酸性。

原理：加入熟石灰Ca(OH)₂，中和酸性后再排放。

🍪 改良土壤

现象：酸性土壤不利于作物生长。

原理：施加熟石灰可以中和土壤酸性。
